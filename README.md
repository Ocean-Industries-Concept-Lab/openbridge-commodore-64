# OpenBridge C64: The Future of Maritime Navigation (circa 1982)

## ⚠️ CRITICAL NOTICE
This project was launched on April 1st, 2026, as an April Fool's joke by Maarten Roest. While the code is functional in its own retro way, it is NOT intended for professional use, actual maritime navigation, or the operation of any vessel larger than a rubber duck.


## Overview
Welcome to OpenBridge C64, the world’s first (and hopefully last) implementation of the OpenBridge Design System for the Commodore 64.
The code is made by Maarten Roest and runs on Commodore 64.
